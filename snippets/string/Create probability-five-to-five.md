title: This method can return a random boolean value that, compared to 0.5, has half the probability of getting a true or false value.
category: String
---

**JavaScript version**

```js
const randomBoolean = () =>  Math.random() >= 0.5
```

**TypeScript version**

```js
const randomBoolean = () =>  Math.random() >= 0.5
```

**Examples**

```js
randomBoolean();
```
